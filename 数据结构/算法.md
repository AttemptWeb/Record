# 算法

## 共性和特点
**计算 == 信息处理**

借助某种工具，遵循一定的规则，以明确的而机械的形式进行
    计算模型 = 计算机 = 信息处理工具

**所谓算法，即特定计算模型下，自在解决特定问题的指令序列**

* 1、输入 待处理的信息（问题）
* 2、输出 进出你的信息（答案）
* 3、正确性 的确可以解决指定的问题
* 4、确定性 任一算法都可以描述为一个由基于组成的序列
* 5、可行性 每一基本操作都可以实现，且在常数时间完成
* 6、有穷性 对于任意输入，经过有穷次操作，都可以得到输出
